In process

Our goal is to accelerate FEM simulationsd to enable efficient parameter estimation and improve realism of simulations. The method is based on:

  • dimensionality analysis and reduction to enable synthetic mesh generation. This enables to enlarge the current dataset.
  • train a deep learning densely connected neural network to be able to learn nodal displacements.
